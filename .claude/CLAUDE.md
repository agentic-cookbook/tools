# Tools

User-facing tools for interacting with the agentic cookbook system.

## Sibling Repositories

This repo depends on two sibling repos at the same directory level:

- **`../cookbook`** — The cookbook itself: principles, guidelines, recipes, workflows, and reference material
- **`../dev-team`** — Multi-agent dev team: agents, rules, scripts, and docs for project discovery, generation, building, and review
